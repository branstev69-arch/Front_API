# Front_API
Pequeña aplicación Flask que muestra login/registro y una UI para gestionar productos consumiendo una API remota.

Requisitos:
- Python 3.8+
- Acceso a red para la API remota

Dependencias (instalar en el entorno del contenedor; pegar y ejecutar en una sola línea):
python3 -m pip install --upgrade pip && python3 -m pip install flask requests

Ejecución:
python app.py
Abrir en el navegador del host:
"$BROWSER" http://localhost:5000
